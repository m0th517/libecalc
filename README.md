# Libecalc
Libecalc is a lets you compare Poker Hands and Ranges.

## Requirements
* UnitTest++ for tests

## Build
```
$ git clone --recursive https://github.com/pandaant/libecalc
$ cd libecalc/dep/libpoker
$ make all
$ cd ../../
$ make all

# if you want to run tests
$ cd test 
$ make all run
```
